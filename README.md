# Custom News 

Retrieves news from custom RSS feeds.

- Devops.com
- DZone.com
- Threatpost.com
- Solar.energy-business-review.com


## Installation

In hubot project repo, run:

`npm install hubot-custom-news --save`

Then add **hubot-custom-news** to your `external-scripts.json`:

```json
[
  "hubot-custom-news"
]
```
